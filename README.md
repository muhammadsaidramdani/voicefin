# Aplikasi VoiceFin  
**Aplikasi Pencatatan Keuangan Berbasis Web dan Android**

---

## 1. Deskripsi Proyek

**VoiceFin** adalah aplikasi pencatatan keuangan yang dikembangkan untuk membantu pengguna dalam mencatat pemasukan dan pengeluaran secara terstruktur. Aplikasi ini mendukung penggunaan **input suara** serta dapat diakses melalui **web** dan **perangkat Android**.

Proyek ini dikembangkan sebagai **tugas perkuliahan**, dengan tujuan mengimplementasikan konsep pengembangan aplikasi modern menggunakan teknologi web dan mobile.

---

## 2. Tujuan Pengembangan

Tujuan dari pengembangan aplikasi VoiceFin adalah:
1. Menerapkan konsep **pengembangan aplikasi berbasis React**.
2. Mengintegrasikan aplikasi web menjadi **aplikasi mobile Android** menggunakan Capacitor.
3. Memberikan solusi sederhana dalam pencatatan keuangan harian.
4. Melatih penerapan **UI/UX sederhana, responsif, dan fungsional**.

---

## 3. Fitur Aplikasi

Fitur yang tersedia pada aplikasi VoiceFin antara lain:
- Pencatatan pemasukan dan pengeluaran
- Input data menggunakan suara
- Riwayat transaksi keuangan
- Tampilan antarmuka yang responsif
- Akses melalui web dan aplikasi Android

---

## 4. Teknologi yang Digunakan

Aplikasi ini dikembangkan menggunakan teknologi berikut:
- **React** (Frontend)
- **Vite** (Build Tool)
- **TypeScript**
- **Capacitor** (Konversi Web ke Android)
- **Tailwind CSS** (Styling)
- **Android Studio** (Build APK Android)
